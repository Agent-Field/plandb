Interactive Playground

Try PlanDB right in your browser. Follow the tutorial or explore on your own.

Welcome to PlanDB Interactive Playground. Type 'help' to see available commands.
$

Step 1: Init

Start by creating a new project. Use the init command.

Type: plandb init "demo"

Step 2: Add Tasks

Now let's add some tasks with dependencies.

Type: plandb add "Setup" --as setup

Step 3: Dependencies

Add a task that depends on the first one.

Type: plandb add "Code" --dep t-setup

Step 4: Go

Ready to work? Claim the next available task.

Type: plandb go

Step 5: Status

Check the graph status.

Type: plandb status --detail
